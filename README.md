# 智慧病房介绍

> **我司是上市公司入股企业， 为大力发展智慧病房业务， 本着开放共赢的模式， 招募合作伙伴(软件商， 硬件商， 代理商等，ISV合作商等)， 特将智慧病房部分功能及界面开源出来。**
>
> **感兴趣的可以线下参观和交流，联系方式：见底部二维码 **





智慧病区全面打通“信息孤岛”现象，基于全面互联互通和深度智能化建设病区信息化集成平台。平台提供的核心功能包括数据采集， 数据缓存， 数据上传， 数据整合， 数据共享， 权限管理， 配置管理等服务。让信息在智慧病区内触手可及，使得信息在医护患周围全息式展开，信息获取不在需要反复登录各类系统。一屏览尽各类信息，最大程度上消除院内信息不对等，并提供患者便捷床旁服务，包括影视娱乐，健康宣教， 床旁点餐等。





## 一、床旁交互终端系统

​		以患者需求为导向，以便捷醒目、互联互通为原则，床旁交互终端系统，让患者不下床就能获取自身恢复情况，获得各类服务，主要包括呼叫、医院科室介绍、花费清单、健康宣教、物联体征、消息查看、个人档案、今日任务模块。患者能在轻松获取更多关注的信息，提升患者住院的体验。让医护人员在床旁就能了解患者的身体情况，通过对接移动护理系统，协助完成更多护理工作。

| 功能       | 介绍                                                         |
| ---------- | ------------------------------------------------------------ |
| 床头信息卡 | HIS系统自动提取并显示患者信息                                |
| 呼叫       | 支持患者发起病呼，并且可对呼叫类型进行选择，支持自动接听护士站呼叫。 |
| 医院科室   | 查看医院及科室介绍，并能对医护点“星”评价                     |
| 花费清单   | 患者查询每日治疗清单、住院费用查询                           |
| 健康宣教   | 智能交互床头屏提供健康宣教观看。                             |
| 消息查看   | 查看消息，消息列表按照时间倒叙排列，不同类型消息：通知、花费、检验、护理使用不同颜色显示，能区分显示已读和未读消息 |
| 个人档案   | 查看体征数据和检验检查报告，以图表式形式显示体征数据，通过对接医院系统获取患者检验结果 |
| 娱乐功能   | 支持安装第三方应用，为患者带来多样的床旁娱乐，内置影音、游戏功能。 |
| 床旁订餐   | 支持对接院内膳食系统或提供订餐系统，为患者提供床旁订餐服务。 |
| 人脸识别   | 支持人脸识别医护人员身份，进入医护界面                       |
| NFC识别    | 支持刷卡识别医护人员身份，进入医护界面                       |
| 患者信息   | 显示患者床号、姓名、性别、诊断、责任医生、责任护士等信息     |
| 医嘱       | 显示患者医嘱内容，可根据长期、临时等维度进行过滤显示         |
| 检验信息   | 显示患者检验报告                                             |
| 检查报告   | 显示患者检查报告，支持影像图片查看                           |
| 体征录入   | 支持在床旁录入体征，支持对接体征采集设备，自动上传体征数据   |
| 住院预存   | 支持床旁交互终端进行住院金缴纳及缴纳查询                     |
| 问卷调查表 | 支持床旁填写院内问卷调查表。                                 |
| 患者任务   | 患者任务推送，区分今日患者已完成未完成任务                   |
| 物联对接   | 对接第三方物联设备。如输液监控， 床垫， 多体征设备等         |

## 五、智能护士站交互系统(护士站大屏)

​		智能护士站交互系统为护士站大屏程序设计的系统。 智慧护士站交互大屏系统作为临床数据和信息处理中枢，拥有自然流畅的操作体验，友好的用户界面及实时的数据处理能力。
​		智慧护士站交互大屏系统通过患者数据信息化，病区状态可视化，有助于减少一线护士工作量，节省时间和资金成本，有助于提升护理质量、效率和安全性，有助于提高患者满意度。

| 功能        | 介绍                                                         |
| ----------- | ------------------------------------------------------------ |
| 护理项目    | 根据院内系统，对医嘱进行关键信息提前，动态更新展示， 有助于提高护理工作质量。 |
| 病患一览    | 对医嘱，护理记录，患者信息多维清洗和重组，以卡片形式呈现，护士对病区情况一屏尽览。 |
| 护士中心    | 显示病区内护士的信息。                                       |
| 智能交班    | 自动提取交班数据信息，生成交班报告。                         |
| 护士排班    | 进行护士排班及展示。                                         |
| 床位分配    | 实时显示病区内床位分配情况。                                 |
| 护理统计    | 根据配置，统计病区的病患信息，床位信息， 护理项目等。        |
| 白板        | 通过电子触摸屏实现及取代传统的护士站白板功能。               |
| 工具        | 根据护士站实际场景，集成一些常见的工具。                     |

## 六、Web后台系统

​		智慧病区后台管理系统为整个智慧病区提供基础服务，管理病区的其他子系统， 操作员通过该系统对病区其他子系统进行设置，人员管理， 物联设备的管理， 护理项目护理事件的维护及基础的数据字典和术语字段维护等。

| 功能        | 介绍                                                         |
| ----------- | ------------------------------------------------------------ |
| 登录        | 根据账密登录系统。                                           |
| 首页        | 首页展示病区整体数据。                                       |
| 菜单管理    | 维护系统菜单，主要功能包含菜单列表，新增菜单，修改菜单，删除菜单 |
| 角色管理    | 维护系统角色，主要功能包含角色列表，新增角色，删除角色，编辑角色，及角色对应的权限。 |
| 操作员管理  | 维护系统操作员，主要包含列表， 新增，编辑及绑卡和解绑卡。    |
| 病区管理    | 查看病区数据，主要包含列表，新增，编辑和删除功能。           |
| 病房管理    | 维护病区内病房信息。                                         |
| 床位管理    | 维护病区床位信息。                                           |
| 床头卡管理  | 动态配置病区床头卡                                           |
| 护理项目    | 维护院内护理项目，用于医嘱数据拆分                           |
| 排班管理    | 维护护士值班情况。                                           |
| 交班管理    | 维护交班情况， apn交班， sbar交班                            |
| 设备管理    | 维护病区内设备数据。                                         |
| 健康宣教    | 维护病区健康宣教内容。                                       |
| 影视娱乐    | 上传一些娱乐影音， 并于病人缓解情绪。                        |
